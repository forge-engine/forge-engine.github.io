Forge Engine

a simple php framework for people who like to build things their way

get started docs github

philosophy

Forge isn’t trying to compete with large frameworks. It’s for developers who want complete control, who enjoy understanding every line of their stack, and who value simplicity over trends. It’s built around a clear idea: use what you need, nothing more.

We built Forge to support real-world projects, not hypothetical features. It’s a personal toolbox that just happens to be open source — no bloat, no dependencies, just the essentials.

install

bash <(curl -Ls https://raw.githubusercontent.com/forge-engine/installer/main/installer.sh)

features

  • no magic – you control everything
  • modular – use only what you need
  • native php views with layout and component support
  • tiny footprint, no dependencies
